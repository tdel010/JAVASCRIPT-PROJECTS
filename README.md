# README

This folder contains assignments and some small projects created for The Tech Academy's JavaScript course.

## Projects of note:

- Calculator
- Pizza Menu
- Tic-Tac-Toe Game

### Calculator
Found inside the "Advanced JavaScript Projects" folder, this is a simple four-function calculator which allows users to add, subtract, multiple, and divide on a web-based app.

### Pizza Menu
This project is a menu used to order pizza by size and toppings. It calculates a price for the order by using the user's input.

### Tic-Tac-Toe Game
This is a web version of the classic game of Tic-Tac-Toe. Users face off against the computer, and there are fun sound effects to liven things up. 

## Other projects
A number of other small projects completed to demonstrate learning of various JavaScript skills can also be found in this repository. Please feel free to browse around!
